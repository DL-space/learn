Advanced link styles using CSS2.1 and CSS3

Some of these effects require Safari or WebKit browser, some require Firefox or Opera. I'll mention the support next to each link.

  • Shows an arrow in right corner if the link is external, all other than IE6
  • Adds shadow, Safari 3+ or Opera 9.6+, degrades gracefully in other browsers
  • Adds background, animated, WebKit or Safari 3+, degrades gracefully in other browsers
  • Changes background, animated, WebKit or Safari 3+, degrades gracefully in other browsers
  • Changes color, shows custom tooltip with the URL, reliably only WebKit, Safari 3+, and IE8 (surprise). Doesn't degrade gracefully but could with some hacks
